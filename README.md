# gmail_roommonitor
Raspberry Piで作るgmailで操作するルームモニターシステム

## 必要機材
- ラズベリーパイ : Raspberry Pi 3 Model B    × 1
- ACアダプタ                                 × 1
- 32GB SDカード                              × 1
- ケース                                     × 1
- カメラ : Raspberry Pi Camera Module V2     × 1
- サーボモータ : SG90                        × 2
- カメラマウント                             × 1
- 温湿度センサ :  AM2320                     × 1
- ダストセンサ :  GROVEダストセンサ(PPD42NS) × 1
- ブレッドボード                             × 1
- 抵抗1KΩ                                   × 5
- ジャンパー線                               × 適量
